## [0.1.0] - 2026-06-12

### Added
- Initial release from Lumex monorepo split
- `anchor-business-server` with SEP-12 KYC and SEP-38 quote endpoints
- `bridge-core` with deposit/withdrawal handlers and MT4 integration
- `shared` package published as `@lumex/shared` to GitHub Packages
- `compliance-sdk` with Chainalysis, Onfido, and Sumsub integrations
- `corridor-configs` for NGN, KES, GHS, ZAR, BRL corridors
- Docker Compose infrastructure for local development
- GitHub Actions CI/CD pipeline
